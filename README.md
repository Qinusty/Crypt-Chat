# Client/Server encrypted chat application

## Client
Modify config.json and add server details.
```
git clone git@github.com:Qinusty/Crypt-Chat.git
cd Crypt-Chat/src
python Client.py
```

## Server
Configure server via server_config.json
```
git clone git@github.com:Qinusty/Crypt-Chat.git
cd Crypt-Chat/src
python Server.py
```
